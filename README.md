# LSM

### Objetivo

Hacer conocer a más gente la LSM

### Descripción

A diferencia de la aplicación ya existente, esta nueva App cuenta con dos nuevos juegos que hacen al usuario aprender la LSM y con una sección de Perfil donde el Usuario podrá ver sus datos.
### Características generales

Usuario
Juego de Mano Interactiva
Juego P2P
Diccionario
